# 🤝 Guide de contribution

Ce projet est un projet interne privé.

---

## Workflow

- Travailler sur des branches séparées
- Ne pas push directement sur main
- Code propre et lisible obligatoire

---

## Règles

- Ne pas exposer de données sensibles
- Ne pas casser la compatibilité du plugin
- Code structuré et maintenable

---

## Pull Requests

Toutes les PR doivent être validées avant merge.
