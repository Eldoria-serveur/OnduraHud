# OnduraHud

Plugin Minecraft Paper permettant d’afficher un HUD dynamique via BossBar avec support de PlaceholderAPI et Oraxen.

---

## ✨ Features
- HUD custom basé sur configuration YAML
- Support PlaceholderAPI
- Support Oraxen glyphs
- Affichage via BossBar
- Formatage automatique des nombres (K / M / B)
- Système de shift / padding pour alignement précis

---

## 📦 Requirements
- Paper 1.21+
- PlaceholderAPI
- Oraxen
- Vault (si économie utilisée dans placeholders)

---

## ⚙️ Installation
1. Build le plugin :
```bash
mvn clean package
```
Placer le jar dans

`/plugins`

Redémarer le serveur.

## 🧾 Commands
 - /ondurahud reload → reload config
